# Driver Learning App (600 Questions - B2 License)

A modern, cross-platform mobile application built with Flutter and Firebase to help Vietnamese users master the 600 official traffic law questions and ace their B-class driving license exam.

## Key Features

* Critical Questions: A dedicated section for high-priority questions that you must answer correctly to pass.
* Smart Error Tracking: Automatically records and categorizes questions you frequently get wrong during mock tests.
* Pro Tips & Tricks: Curated "cheat sheets" and memory tips to help you choose the correct answers quickly.
* Realistic Mock Exams: Experience the actual exam pressure with a randomized set of questions.
* Readiness Assessment: An evaluation that analyzes your test history and tells you when you are ready.

## Tech Stack

* Frontend: Flutter (Dart)
* Backend/Database: Firebase Firestore
* Authentication: Firebase Auth

## Getting Started

1. Clone the repository:
   git clone https://github.com/TrucNgo2303/LearningDriver

2. Navigate to the project folder:
   cd learning_driver_app

3. Install dependencies:
   flutter pub get

4. Run the app:
   flutter run
